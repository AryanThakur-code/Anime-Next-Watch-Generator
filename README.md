# Anime-Next-Watch-Generator

🎌 Anime Next Watch Generator
📌 Description

Anime Next Watch Generator is a web application designed for users who struggle to decide what anime to watch next. The app fetches top-trending anime using a public API and allows users to explore, filter, and discover anime based on their preferences.

This project demonstrates API integration, dynamic UI rendering, and the use of JavaScript higher-order functions.

🚀 Features
🔥 Trending Anime List
Displays currently popular/trending anime fetched from the API
🎯 Genre-Based Filtering
Filter anime by genres like Action, Romance, Comedy, etc.
⭐ Top Rated Filter
View anime sorted by highest ratings
🔍 Search Functionality
Search anime by name (using keywords)
📖 Expandable Synopsis Card
Click to view full anime description/synopsis
⏳ Loading State Handling
Displays loading indicators while fetching data
🌐 API Used

This project uses the Jikan API (MyAnimeList API)

🔗 https://docs.api.jikan.moe

🛠️ Tech Stack
HTML
CSS (or Tailwind / Bootstrap - optional)
JavaScript (Vanilla JS)
Fetch API
⚙️ How It Works
Fetches anime data from Jikan API using fetch()
Displays anime dynamically on the webpage
Uses JavaScript higher-order functions like:
map() → render UI
filter() → genre filtering
sort() → top-rated sorting
Updates UI based on user interaction
